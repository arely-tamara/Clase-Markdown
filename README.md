# Clase-Markdown
Github: Plataforma en la nube diseñada para desarrolladores, permite almacenar, gestionar y compartir código fuente utilizando el sistema de control de versiones de Git. Funciona como una especie de red social para programadores, facilitando la colaboracón, el seguimiento de cambios y desarollo de proyectos de softwares tanto publico como privados.
MARKDOWN: ES UN LENGUAJE DE MERCADO LIGERO. EN TERMINOS SIMPLES, ES UNA FORMA DE ESCRIBIR TEXTOCON FORMATO.
SALTOS DE LINEA:
Si quieres separar dos parrafos, debes presionar enter dos veces. Esto deja un espacio en blanco entre los bloques de texto.

Parrafo ejemplo de uso de comando.

Si quieres que el texto baje a la siguiente  linea sin dejar un espacio en blanco, debes poner dos espacios en blanco al final de la frase y luego presionar enter
